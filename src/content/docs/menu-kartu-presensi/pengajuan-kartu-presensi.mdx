Pengajuan Kartu adalah proses formal untuk mendapatkan kartu presensi RFID baru. Setiap pengajuan akan melalui tahapan review, persetujuan, pembuatan, hingga distribusi kartu kepada pemohon.

## Mengajukan Kartu Baru

<Steps>

1. **Login ke Sistem**

   Masuk menggunakan username dan password Anda

2. **Buka Menu Pengajuan Kartu**

   Navigasi ke **Kartu Presensi â†’ Pengajuan Kartu Presensi**

3. **Klik Tombol Ajukan**

   Klik **"+ Ajukan Kartu Baru"** di pojok kanan atas

4. **Isi Form Pengajuan**

   Lengkapi informasi berikut:
   - **Jenis Pengajuan** (wajib)
     - â—‹ Kartu Baru (pertama kali)
     - â—‹ Pengganti Hilang
     - â—‹ Pengganti Rusak

   - **Alasan Pengajuan** (wajib jika pengganti)
     - Jelaskan kronologi kehilangan/kerusakan

   - **Upload Foto** (wajib untuk kartu baru)
     - Format: JPG/PNG
     - Ukuran: Maksimal 2 MB
     - Background: Merah untuk siswa, Biru untuk pegawai
     - Pas foto formal (tidak selfie)

   - **Konfirmasi Data Diri**
     - Nama, NIS/NIP, Kelas/Jabatan (auto-filled)
     - Nomor WhatsApp aktif
     - Alamat lengkap

5. **Review & Konfirmasi**
   - Periksa kembali semua data
   - Baca syarat dan ketentuan
   - Centang "Saya setuju dengan biaya Rp 15.000"

6. **Submit Pengajuan**

   Klik **"Kirim Pengajuan"**

   **Status**: Pending

7. **Tunggu Persetujuan**
   - Admin akan mereview dalam 1x24 jam
   - Anda akan menerima notifikasi via WhatsApp dan sistem

</Steps>

<Aside type="note">
  Pastikan foto yang diupload sudah sesuai ketentuan. Pengajuan dengan foto
  tidak sesuai akan **ditolak** dan harus mengajukan ulang.
</Aside>

## Registrasi Nomor RFID

Setelah pengajuan disetujui, admin akan meregistrasi nomor RFID unik untuk kartu Anda.

### Proses Registrasi (Admin)

<Steps>

1. **Scan Kartu Kosong**

   Admin scan kartu RFID kosong dengan reader

2. **Sistem Baca Nomor Unik**

   Nomor RFID otomatis terbaca (contoh: 0AB3D5E7F1)

3. **Link dengan Data User**

   Nomor RFID dihubungkan dengan data siswa/pegawai di database

4. **Aktivasi Kartu**

   Kartu diaktifkan dan siap digunakan setelah cetak

5. **Validasi**

   Admin test tap kartu untuk memastikan sudah terhubung

</Steps>

<Aside type="tip" title="Untuk Admin">
  Gunakan fitur **"Registrasi RFID Massal"** untuk meregistrasi banyak kartu
  sekaligus, misalnya untuk siswa baru angkatan baru.
</Aside>

  Nomor WhatsApp tidak valid **Aksi**: Status â†’ **Ditolak** + Alasan penolakan
  **Notifikasi**: User diminta mengajukan ulang
</TabItem>

  <TabItem label="Pending Review">

## Status Pengajuan

**Pengajuan baru masuk, menunggu review**

**Yang Terjadi**:

- Pengajuan masuk ke antrian admin
- Admin akan review dalam 1x24 jam kerja
- User dapat membatalkan pengajuan jika berubah pikiran

**Yang Bisa Dilakukan User**:

- Cek status di dashboard
- Batalkan pengajuan
- Belum bisa ambil kartu

**Estimasi Waktu**: Maksimal 1 hari kerja

## Kartu Hilang/Rusak

### Mengajukan Kartu Pengganti

<Steps>

1. **Lapor Kehilangan/Kerusakan**

   Segera lapor ke bagian TU atau operator

2. **Admin Nonaktifkan Kartu Lama**

   Kartu lama dinonaktifkan untuk mencegah penyalahgunaan

3. **Ajukan Kartu Pengganti**
   - Pilih jenis: **"Pengganti Hilang"** atau **"Pengganti Rusak"**
   - Jelaskan kronologi kehilangan/kerusakan
   - Upload bukti (jika ada): foto kartu rusak, surat kehilangan, dll

4. **Upload Foto Baru**

   Foto yang sama dengan kartu sebelumnya (sudah tersimpan di sistem)

5. **Proses Sama dengan Kartu Baru**
   - Review â†’ Diproses â†’ Selesai â†’ Diserahkan
   - Biaya sama: **Rp 15.000**
   - Nomor RFID baru akan diregistrasi

</Steps>

<Aside type="caution">
  **Penting**: Kartu lama yang hilang akan **langsung dinonaktifkan** begitu
  Anda lapor. Jika kemudian ketemu, kartu lama **tidak dapat diaktifkan
  kembali**. Anda tetap harus menggunakan kartu pengganti yang baru.
</Aside>

## Troubleshooting

### Pengajuan ditolak, kenapa?

**Penyebab Umum**:

- Foto tidak sesuai ketentuan (selfie, background salah)
- Data tidak lengkap atau tidak valid
- Sudah ada pengajuan aktif yang belum selesai

**Solusi**:

- Baca alasan penolakan di notifikasi
- Perbaiki sesuai feedback admin
- Ajukan ulang dengan data yang benar

### Sudah 5 hari tapi kartu belum selesai?

**Kemungkinan**:

- Antrian panjang (periode ramai seperti awal tahun ajaran)
- Printer/alat cetak bermasalah
- Menunggu stok kartu kosong

**Solusi**:

- Hubungi admin/operator untuk konfirmasi
- Cek status di dashboard untuk update terbaru
- Bersabar, admin akan infokan jika ada kendala

### Notifikasi WhatsApp tidak masuk?

**Penyebab**:

- Nomor WhatsApp salah/tidak aktif
- WhatsApp diblokir
- Gateway WhatsApp madrasah down

**Solusi**:

- Update nomor WhatsApp di profil
- Cek status via dashboard (tidak bergantung WA saja)
- Hubungi admin untuk re-send notifikasi

### Kartu tidak berfungsi saat tap?

**Penyebab**:

- Kartu belum diaktifkan oleh admin
- Chip RFID rusak
- Reader bermasalah

**Solusi**:

- Test di reader lain untuk isolir masalah
- Konfirmasi ke admin apakah nomor RFID sudah teregistrasi
- Jika chip rusak, ajukan penggantian

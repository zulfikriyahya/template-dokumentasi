<Steps>

1. **Akses Menu Informasi**

2. **Lihat Data Saat Ini**

   Sistem akan menampilkan informasi sekolah yang tersimpan saat ini. Periksa data yang perlu diperbarui.

3. **Edit Informasi**

   Klik tombol **"Edit"** atau **"Ubah"** pada bagian atas halaman untuk masuk ke mode editing.

   Lengkapi atau perbarui field yang diperlukan:

   Untuk mengunggah atau mengubah logo:

   - Klik area upload atau tombol **"Pilih File"**
   - Pilih file gambar (PNG/JPG/SVG)
   - Pastikan ukuran file tidak melebihi 2MB
   - Preview akan muncul setelah file dipilih

   <Aside type="note">
     Disarankan menggunakan logo dengan rasio 1:1 (square) untuk hasil terbaik
     di berbagai tampilan.
   </Aside>

   Periksa kembali seluruh data yang telah diisi:

   - Validasi data yang diinput

## ğŸ–¼ï¸ Panduan Upload Logo

### Spesifikasi Logo

<Card title="ğŸ“ Persyaratan Teknis" icon="approve-check">

**Format File:**

- PNG (Recommended) - Mendukung transparansi
- JPG/JPEG - Untuk foto atau gambar kompleks
- SVG - Vector, resolusi tak terbatas

**Ukuran File:**

- Maksimal: 2 MB
- Recommended: 500 KB - 1 MB

**Dimensi:**

- Minimum: 200 x 200 px
- Recommended: 512 x 512 px atau 1024 x 1024 px
- Rasio: 1:1 (Square)

**Background:**

- PNG dengan transparansi (untuk fleksibilitas)
- Atau background putih untuk logo berwarna

</Card>

### Tips Upload Logo

<Aside type="tip" title="Best Practices">

1. **Gunakan logo beresolusi tinggi** untuk hasil terbaik di berbagai ukuran layar
2. **Siapkan 2 versi** - satu untuk light mode, satu untuk dark mode
3. **Test tampilan** setelah upload di berbagai halaman sistem
4. **Compress file** jika ukuran terlalu besar tanpa mengurangi kualitas
5. **Gunakan format PNG** dengan background transparan untuk fleksibilitas maksimal

</Aside>

## â“ FAQ - Pertanyaan Umum

<details>
<summary>**Apakah perubahan informasi langsung terlihat di sistem?**</summary>

Ya, setelah menyimpan perubahan, informasi akan langsung terupdate di seluruh sistem. Namun untuk beberapa cache, mungkin perlu refresh halaman (F5) untuk melihat perubahan.

</details>

<details>
<summary>**Bagaimana jika logo tidak muncul setelah diupload?**</summary>

Pastikan:

1. Format file sudah benar (PNG/JPG/SVG)
2. Ukuran file tidak melebihi 2MB
3. File tidak corrupt atau rusak
4. Clear cache browser dan refresh halaman
5. Hubungi admin jika masalah berlanjut

</details>

<details>
<summary>**Siapa yang bisa mengubah informasi sekolah?**</summary>

Hanya **Super Admin** dan **Administrator** yang memiliki hak akses untuk mengubah informasi sekolah. Operator dan user biasa hanya bisa melihat (read-only).

</details>

<details>
<summary>**Apakah ada log perubahan data?**</summary>

Ya, sistem mencatat setiap perubahan data termasuk waktu, user yang melakukan perubahan, dan data apa yang diubah untuk keperluan audit.

</details>

<details>
<summary>**Bagaimana cara mengganti logo untuk dark mode?**</summary>

Upload logo dengan warna yang kontras untuk background gelap di field "Logo Dark Mode". Sistem akan otomatis menggunakan logo yang sesuai berdasarkan theme yang dipilih user.

</details>
